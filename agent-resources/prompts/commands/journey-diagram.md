# Journey Diagram Command Prompt

Generate user journey and UX diagrams as Mermaid diagram-as-code.

## Required behavior

- Read PRD, routes, frontend components, DESIGN.md, telemetry hints, and user input.
- Include primary journey map, user flow, decision/state diagram, funnel/friction map, sequence diagram, and instrumentation event flow when useful.
- Identify missing states: loading, empty, error, disabled, permission denied, success, retry, timeout, and recovery.
- For every diagram include source evidence, assumptions, and update trigger.
