# Project Diagram Pack Command Prompt

Choose and generate the smallest useful diagram set for the current SDLC stage.

## Diagram decision matrix

- Product/PRD: user flow, journey map, dependency map.
- Architecture/API/backend: context, container/component, deployment, sequence, trust boundary.
- Frontend/UI: route map, component tree, state transition, interaction sequence.
- Data/SQL: ERD, lineage, pipeline DAG, DQ control flow, reconciliation flow, privacy boundary.
- Release/runbook: gate flow, rollback flow, incident escalation flow.

Use Mermaid by default and never invent unsupported nodes or edges.
