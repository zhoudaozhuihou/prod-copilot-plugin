# SQL Dialect Translation Prompt

Translate SQL between PostgreSQL, Oracle, BigQuery, MaxCompute/ODPS, MySQL, SQL Server, Snowflake, Databricks/Spark SQL, and Hive while preserving business semantics.

Always review data types, date/time functions, string functions, null semantics, window functions, CTEs, MERGE/UPSERT, temporary tables, sequences/identity columns, partition syntax, quoting, arrays/structs, UDFs, and unsupported functions.
