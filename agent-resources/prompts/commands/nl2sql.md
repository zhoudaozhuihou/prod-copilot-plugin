# NL2SQL Command Prompt

Convert natural language business questions into safe SQL.

Supported dialects: PostgreSQL, Oracle, BigQuery, MaxCompute/ODPS, MySQL, SQL Server, Snowflake, Databricks/Spark SQL, Hive.

Required output: business question, dialect, assumptions, required schema context, metric/dimension/grain definition, SQL, join/filter explanation, validation SQL, DQ/reconciliation checks, performance notes, privacy/access risks, open questions, next command.
