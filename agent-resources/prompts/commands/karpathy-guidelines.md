# Karpathy Guidelines Command

Use this command to apply the `karpathy-guidelines` skill to the current task.

## Input

User task, selected files, attachments, repo context, policy packs, and applicable domain skills.

## Process

1. State assumptions and ambiguity.
2. Choose the minimum safe scope.
3. Identify surgical files/artifacts to touch.
4. Define success criteria and verification checks.
5. Produce the requested artifact or review.
6. End with the next command.

## Output

Task interpretation → assumptions → minimal plan → verification checks → output → risks/open questions → next command.
