# /code-graph

Generate a GitNexus-inspired repository knowledge graph artifact.

Inputs: repository scan, active editor, selected code, attachments, optional GitNexus CLI/MCP output, and user request.

Output must include modules, functional clusters, entry points, execution flows, dependency graph, API/data/UI links, risky coupling, Mermaid diagrams, and next commands.

Do not invent graph edges. Mark uncertain edges as inferred.
