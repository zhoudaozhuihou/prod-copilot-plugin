# GitNexus-inspired Patterns

GitNexus indexes a repo into a code knowledge graph and exposes MCP-style operations such as repo query, context, impact, detect changes, graph queries, and map generation.

This project does not embed GitNexus itself. It provides prompt/skill-compatible commands that can consume GitNexus output when users attach it or expose it through MCP-enabled agents.

Recommended artifact sections:
- Module graph
- Functional clusters
- Entry points
- Execution flows
- Dependency edges
- Blast radius
- Change detection
- Agent navigation queries
- Mermaid architecture map
