# Spring Boot Code Reading Checklist

- `@RestController` / `@Controller`
- Class-level `@RequestMapping`
- Method-level mapping annotations
- `@PathVariable`, `@RequestParam`, `@RequestBody`, `@RequestHeader`
- Bean Validation: `@Valid`, `@NotNull`, `@NotBlank`, `@Size`, `@Pattern`, `@Min`, `@Max`, `@Email`
- Security: `@PreAuthorize`, `@Secured`, `@RolesAllowed`
- Exception mapping: `@ControllerAdvice`, `@ExceptionHandler`
- Service calls and branch behavior
- Persistence side effects
