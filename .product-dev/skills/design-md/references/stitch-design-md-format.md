# Stitch-compatible DESIGN.md Format

Use these sections:

1. Visual Theme & Atmosphere — mood, density, visual philosophy.
2. Color Palette & Roles — semantic names, hex values when known, usage roles.
3. Typography Rules — font families, hierarchy, weights, line-height, letter spacing.
4. Component Stylings — buttons, inputs, cards, tables, navigation, states.
5. Layout Principles — grid, spacing scale, breakpoints, whitespace.
6. Depth & Elevation — surfaces, shadows, overlays, z-index conventions.
7. Do's and Don'ts — guardrails and anti-patterns.
8. Responsive Behavior — breakpoints, collapsing strategy, touch targets.
9. Agent Prompt Guide — compact prompts agents can reuse to generate matching UI.
