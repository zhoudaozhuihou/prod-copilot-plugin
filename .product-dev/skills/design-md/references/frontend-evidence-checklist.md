# Frontend Evidence Checklist

Look for:

- package.json dependencies: React, Vue, Angular, MUI, Ant Design, Tailwind, Chakra, styled-components, Emotion.
- CSS variables in `:root`, theme providers, token files, Tailwind config.
- Layout primitives: AppShell, Layout, Header, Sidebar, Grid, Container.
- Component states: hover, focus-visible, active, disabled, loading, error, empty.
- Typography imports, font-family, heading classes, text components.
- Color tokens, semantic roles, status colors, chart palettes.
- Responsive patterns: breakpoints, mobile nav, collapsible sidebars, touch targets.
- Accessibility indicators: aria labels, focus rings, keyboard interactions, contrast notes.
