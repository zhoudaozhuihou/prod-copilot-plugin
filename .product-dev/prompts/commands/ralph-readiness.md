# Ralph Readiness Command Prompt

Review whether the project is ready for Ralph-style loop execution.

## Check

- PRD completeness
- prd.json schema and story size
- dependency ordering
- acceptance criteria verifiability
- quality commands available
- progress.txt append-only memory
- AGENTS.md / CLAUDE.md reusable pattern guidance
- branchName and git readiness
- stop condition and COMPLETE behavior

Return a readiness decision: Approved, Approved with Conditions, or Not Ready.
