# Company Rollout Plan

## Phase 1: Local MVP

- Enable `/brainstorm`, `/feature`, `/prd`, `/journey`, `/review`.
- Output artifacts to local `docs/`.
- Pilot with one frontend-heavy product.

## Phase 2: Diff-driven SDLC

- Enable `/diff`, `/test`, `/release`.
- Require PRD update proposal for meaningful code changes.
- Add quality gates to pull request checklist.

## Phase 3: Enterprise integration

- Add Jira story creation.
- Add Confluence publishing.
- Add GitHub/GitLab PR comment integration.
- Add SonarQube findings import.
- Add API Gateway and metadata service context.

## Phase 4: AI Delivery OS

- Central prompt registry.
- Team-specific policy packs.
- Audit log.
- Approval workflow.
- Management dashboard.
